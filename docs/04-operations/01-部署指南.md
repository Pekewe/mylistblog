# 部署指南

> 版本：1.0.0 | 最后更新：2026-05-24
> 目标系统：CentOS 9

---

## 1. 环境要求

| 项目 | 要求 |
|------|------|
| 操作系统 | CentOS 9（或其他 Linux 发行版） |
| Node.js | 18+ 或 20+ LTS |
| 内存 | 最低 256MB，推荐 512MB+ |
| 磁盘 | 100MB（不含上传文件） |
| 域名 | 可选（无域名可用 IP 访问） |

## 2. 安装 Node.js

```bash
# 安装 Node.js 20 LTS
curl -fsSL https://rpm.nodesource.com/setup_20.x | bash -
dnf install -y nodejs

# 验证
node --version   # v20.x
npm --version    # 10.x
```

## 3. 上传项目文件

```bash
# 创建项目目录
mkdir -p /var/www/mylistblog

# 上传文件（在本地执行）
scp -r /本地项目路径/* root@你的服务器IP:/var/www/mylistblog/
```

## 4. 安装依赖

```bash
cd /var/www/mylistblog/server
npm install --production
```

## 5. 安装 PM2（进程管理）

```bash
npm install -g pm2

cd /var/www/mylistblog
pm2 start server/app.js --name mylistblog

# 开机自启
pm2 startup
# 执行上述命令后，复制输出的命令执行

# 保存进程列表
pm2 save
```

## 6. 配置 Nginx（可选但推荐）

```bash
dnf install -y nginx
```

创建配置文件 `/etc/nginx/conf.d/mylistblog.conf`：

```nginx
server {
    listen 80;
    server_name your-domain.com;  # 改成你的域名或IP

    client_max_body_size 10m;

    location / {
        proxy_pass http://127.0.0.1:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_cache_bypass $http_upgrade;
    }
}
```

```bash
# 启动 Nginx
systemctl enable nginx
systemctl start nginx

# 放行防火墙
firewall-cmd --permanent --add-service=http
firewall-cmd --reload
```

## 7. 配置 HTTPS（可选）

```bash
dnf install -y certbot python3-certbot-nginx
certbot --nginx -d your-domain.com
```

## 8. 安全设置

```bash
# 设置环境变量覆盖默认密钥
export SESSION_SECRET="你的随机长字符串"
pm2 restart mylistblog

# 修改默认密码
# 登录后台 → 站点设置 → 修改管理员密码
```

## 9. 验证部署

```bash
# 检查进程状态
pm2 status

# 测试 API
curl http://localhost:3000/api/lists

# 通过域名或IP访问
curl http://your-domain.com/index.html
```
