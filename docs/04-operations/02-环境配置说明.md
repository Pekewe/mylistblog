# 环境配置说明

> 版本：1.0.0 | 最后更新：2026-05-24

---

## 1. 开发环境

| 项目 | 配置 |
|------|------|
| Node.js | 18+ / 20+ |
| 编辑器 | 任意（推荐 VS Code） |
| 操作系统 | Windows / macOS / Linux 均可 |

### Windows 开发环境

1. 从 https://nodejs.org/ 下载 LTS 版本安装
2. 使用 `start.bat` 启动项目
3. 浏览器访问 `http://localhost:3000`

### macOS / Linux 开发环境

```bash
# 安装 Node.js（推荐使用 nvm）
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install 20

# 启动
cd 项目目录
node server/app.js
```

## 2. 生产环境

### 硬件要求

| 配置项 | 最低 | 推荐 |
|--------|------|------|
| CPU | 1核 | 2核 |
| 内存 | 256MB | 512MB |
| 磁盘 | 1GB | 10GB |
| 带宽 | 1Mbps | 5Mbps |

### 软件要求

| 软件 | 版本 | 用途 |
|------|------|------|
| Node.js | 20 LTS | 运行环境 |
| Nginx（可选） | 最新稳定版 | 反向代理 + HTTPS |
| pm2（可选） | 最新版 | 进程管理 |

## 3. 端口说明

| 端口 | 用途 | 说明 |
|------|------|------|
| 3000 | Express 服务 | 后端 API + 静态文件 |
| 80 | HTTP（有 Nginx 时） | Nginx |
| 443 | HTTPS（有 Nginx + Certbot 时） | Nginx |

## 4. 环境变量

| 变量名 | 默认值 | 说明 |
|--------|--------|------|
| PORT | 3000 | 服务端口 |
| SESSION_SECRET | mylistblog-default-secret-key | Session 加密密钥 |
