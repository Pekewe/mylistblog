## 2. 表结构

### 2.1 lists（清单主表）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK AUTOINCREMENT | 清单编号 |
| title | TEXT | NOT NULL | 清单名称 |
| product_code | TEXT | DEFAULT '' | 商品編號 |
| category | TEXT | DEFAULT '' | 分类名称 |
| author | TEXT | DEFAULT '' | 作者（发布人显示名称） |
| image | TEXT | DEFAULT '' | 封面图片路径/URL |
| status | TEXT | DEFAULT 'published', CHECK('published','draft','pending') | 发布状态 |
| created_at | TEXT | DEFAULT datetime('now','localtime') | 创建时间 |
| updated_at | TEXT | DEFAULT datetime('now','localtime') | 更新时间 |

### 2.2 list_items（清单内容条目）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK AUTOINCREMENT | 条目编号 |
| list_id | INTEGER | NOT NULL, FK → lists(id) | 所属清单 |
| sort_order | INTEGER | DEFAULT 0 | 排序序号 |
| content | TEXT | NOT NULL | 条目内容 |

### 2.3 list_links（清单相关链接）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK AUTOINCREMENT | 链接编号 |
| list_id | INTEGER | NOT NULL, FK → lists(id) | 所属清单 |
| title | TEXT | NOT NULL | 链接显示名称 |
| url | TEXT | NOT NULL | 链接 URL |

### 2.4 categories（分类）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK AUTOINCREMENT | 分类编号 |
| name | TEXT | NOT NULL UNIQUE | 分类名称 |
| created_at | TEXT | DEFAULT datetime('now','localtime') | 创建时间 |

### 2.5 messages（留言）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK AUTOINCREMENT | 留言编号 |
| name | TEXT | NOT NULL | 留言者昵称 |
| content | TEXT | NOT NULL | 留言内容 |
| status | TEXT | DEFAULT 'pending', CHECK('pending','approved','rejected') | 审核状态 |
| reply_content | TEXT | DEFAULT '' | 管理员回复内容 |
| reply_at | TEXT | — | 回复时间 |
| reply_by | TEXT | DEFAULT '' | 回复人（管理员用户名） |
| created_at | TEXT | DEFAULT datetime('now','localtime') | 创建时间 |

### 2.6 publishers（发布人账号）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK AUTOINCREMENT | 发布人编号 |
| username | TEXT | NOT NULL UNIQUE | 登录用户名 |
| password | TEXT | NOT NULL | bcrypt 哈希密码 |
| display_name | TEXT | DEFAULT '' | 显示名称（清单作者名） |
| status | TEXT | DEFAULT 'active', CHECK('active','disabled') | 账号状态 |
| created_at | TEXT | DEFAULT datetime('now','localtime') | 创建时间 |
| updated_at | TEXT | DEFAULT datetime('now','localtime') | 更新时间 |

### 2.7 settings（站点设置）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| key | TEXT | PK | 设置项名称 |
| value | TEXT | NOT NULL | 设置项值 |

### 2.8 admin_users（管理员账号）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK AUTOINCREMENT | 管理员编号 |
| username | TEXT | NOT NULL UNIQUE | 登录用户名 |
| password | TEXT | NOT NULL | bcrypt 哈希密码 |
| created_at | TEXT | DEFAULT datetime('now','localtime') | 创建时间 |

## 3. 索引

| 索引名 | 表 | 字段 | 用途 |
|--------|-----|------|------|
| idx_lists_status | lists | status | 按状态筛选 |
| idx_lists_category | lists | category | 按分类筛选 |
| idx_lists_created | lists | created_at | 按时间排序 |
| idx_items_list | list_items | list_id | 按清单查询条目 |
| idx_links_list | list_links | list_id | 按清单查询链接 |
| idx_messages_status | messages | status | 按状态筛选留言 |
| idx_messages_created | messages | created_at | 按时间排序留言 |
| (自动) | publishers | username | UNIQUE 约束自动建 |
| (自动) | categories | name | UNIQUE 约束自动建 |
| (自动) | admin_users | username | UNIQUE 约束自动建 |

## 4. 数据量评估

| 表 | 日增量（预估） | 年增量 | 备注 |
|-----|---------------|--------|------|
| lists | 0-3 条 | ~1000 条 | 个人站点，发布频率低 |
| list_items | 0-20 条 | ~5000 条 | 每条清单 2-8 条内容 |
| list_links | 0-5 条 | ~1500 条 | 每条清单 0-3 个链接 |
| messages | 0-5 条 | ~1000 条 | 留言频率低 |
| publishers | 偶尔 | — | 由管理员手动创建 |
| categories | 偶尔 | — | 由管理员手动创建 |

SQLite 在十万级数据量下性能良好，当前设计无需考虑分库分表。
