# 快速入门教程

> 版本：1.0.0 | 最后更新：2026-05-24

---

## 5分钟上手

### 第一步：启动服务器

```bash
# Windows
双击 start.bat

# 或命令行
cd 项目目录
node server/app.js
```

看到以下输出即启动成功：
```
服务器已启动: http://localhost:3000
访问 http://localhost:3000/index.html 查看前台
访问 http://localhost:3000/admin.html 管理后台
```

### 第二步：访问前台

浏览器打开 **http://localhost:3000/index.html**

你会看到：
- 顶部：站点名称 + 导航菜单（首页/分类/搜索/发布/留言/关于）
- 中间：3 条示例清单列表
- 底部：页脚版权信息

### 第三步：登录后台

浏览器打开 **http://localhost:3000/admin.html**

输入默认账号：
| 用户名 | 密码 |
|--------|------|
| admin | admin123 |

登录后你会看到后台管理面板，包含 5 个标签页：
- 清单管理
- 留言管理
- 发布人管理
- 分类管理
- 站点设置

### 第四步：发布一条清单

1. 打开发布页 **http://localhost:3000/lt-post.html**
2. 输入发布人账号：
   | 用户名 | 密码 |
   |--------|------|
   | publisher1 | pub123 |
3. 填写清单名称、选择分类、输入内容
4. 点击「提交清单」

发布成功后，清单状态为「待审核」。回到后台「清单管理」→ 点击「通过」→ 清单出现在首页。

### 第五步：体验留言

1. 打开留言板 **http://localhost:3000/lt-bbs.html**
2. 输入昵称和内容，提交留言
3. 回到后台「留言管理」→ 点击「通过」→ 留言出现在留言板
