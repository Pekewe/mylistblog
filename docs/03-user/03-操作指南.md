# 操作指南（管理员）

> 版本：1.0.0 | 最后更新：2026-05-24

---

## 登录后台

访问 `http://你的域名/admin.html` 或 `http://localhost:3000/admin.html`

输入管理员账号密码登录。

## 清单管理

### 查看清单列表

登录后默认进入「清单管理」标签页。表格展示所有清单（含全部状态），每页 100 条。

### 搜索清单

在搜索框输入关键词，支持搜索：
- 清单标题
- 分类
- 作者
- 状态（如 `pending`、`published`）
- 清单内容

### 审核清单

前台发布人提交的清单默认为「待审核」状态（橙色）。点击列表中的「通过」按钮，该清单状态变为「已发布」，出现在前台首页。

### 新增清单

点击「+ 新增清单」按钮，填写表单直接创建清单（管理员创建不用审核，直接发布）。

### 编辑清单

点击清单行的「编辑」按钮，弹出编辑窗口，修改后保存。

### 删除清单

点击清单行的「删除」按钮，确认后删除。

## 留言管理

### 查看留言列表

点击「留言管理」标签页，表格展示所有留言，每页 100 条。

留言状态：
| 状态 | 颜色 | 说明 |
|------|------|------|
| 待审核 | 橙色 | 新提交的留言，需要审核 |
| 已通过 | 绿色 | 已通过，前台可见 |
| 已拒绝 | 红色 | 已拒绝，前台不可见 |

### 审核留言

- 点击「通过」→ 留言出现在前台留言板
- 点击「拒绝」→ 留言不显示

### 回复留言

点击留言行的「回复」按钮，输入回复内容。回复后前台会在留言下方显示，用不同底色和左边线区分。

## 发布人管理

### 新增发布人

1. 点击「发布人管理」标签页
2. 点击「+ 新增发布人」
3. 填写用户名、密码、显示名称
4. 点击保存

> 显示名称会作为清单的「作者」字段

### 编辑发布人

点击发布人行的「编辑」按钮，可修改显示名称、密码、状态。

### 启用/禁用

- 禁用后，该发布人无法提交清单
- 启用后恢复

## 分类管理

### 新增分类

1. 点击「分类管理」标签页
2. 点击「+ 新增分类」
3. 输入分类名称
4. 点击添加

### 删除分类

点击分类行的「删除」按钮。注意：删除分类不会影响已使用该分类的清单。

## 站点设置

### 修改站点名称/简介

1. 点击「站点设置」标签页
2. 修改站点名称和简介
3. 点击保存设置

修改后刷新前台页面即可看到效果。

### 修改管理员密码

在站点设置页面的「新密码」框中输入新密码，保存设置即可。
