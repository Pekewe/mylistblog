# 用户需求规格说明书（PRD）

> 版本：1.0.0 | 最后更新：2026-05-24

---

## 1. 项目背景

个人清单分享站点。管理员手动创建发布人，发布人提交清单，访客浏览。所有内容经过管理员审核后公开。无需用户注册，发布人由管理员分配。

## 2. 用户角色

### 2.1 访客

| 属性 | 说明 |
|------|------|
| 身份 | 未登录的普通访问者 |
| 权限 | 浏览、搜索、留言 |
| 账号 | 不需要 |

### 2.2 发布人

| 属性 | 说明 |
|------|------|
| 身份 | 由管理员创建的合法发布者 |
| 权限 | 发布清单（需验证账号密码） |
| 账号 | 管理员手动创建 |

### 2.3 管理员

| 属性 | 说明 |
|------|------|
| 身份 | 站点运营者 |
| 权限 | 后台全部功能 |
| 账号 | 系统内置 |

## 3. 功能需求

### 3.1 前台功能需求

| 编号 | 需求名称 | 优先级 | 描述 |
|------|----------|--------|------|
| F01 | 首页清单列表 | P0 | 分页展示已发布的清单，每页30条，按时间倒序 |
| F02 | 分类浏览 | P0 | 按分类筛选清单，展示分类下已发布的清单 |
| F03 | 关键词搜索 | P0 | 搜索已发布清单的标题、分类、作者 |
| F04 | 发布清单 | P0 | 需验证发布人账号密码，提交后为待审核状态 |
| F05 | 留言提交 | P0 | 访客无需登录即可提交留言，提交后为待审核 |
| F06 | 留言展示 | P1 | 分页展示已审核的留言（含管理员回复） |
| F07 | 图片上传 | P1 | 支持本地上传和 URL 外链两种图片方式 |
| F08 | 商品编号 | P1 | 发布清单时可填写商品编号字段 |
| F09 | 关于页面 | P1 | 展示站点介绍和功能说明 |

### 3.2 后台管理需求

| 编号 | 需求名称 | 优先级 | 描述 |
|------|----------|--------|------|
| B01 | 管理员登录 | P0 | 用户名+密码验证，session 24小时 |
| B02 | 清单管理列表 | P0 | 表格展示所有清单，分页100条/页 |
| B03 | 后台清单搜索 | P0 | 搜索标题/分类/作者/状态/内容 |
| B04 | 清单编辑/删除 | P0 | 管理员可编辑和删除任意清单 |
| B05 | 清单审核 | P0 | 将待审核清单设为已发布 |
| B06 | 留言管理列表 | P0 | 表格展示所有留言 |
| B07 | 留言审核 | P0 | 通过/拒绝留言 |
| B08 | 留言回复 | P0 | 管理员回复留言 |
| B09 | 发布人管理 | P0 | 新增/编辑/启用/禁用发布人 |
| B10 | 分类管理 | P1 | 新增/删除分类 |
| B11 | 站点设置 | P1 | 修改站点名称、简介、管理员密码 |

### 3.3 非功能性需求

| 编号 | 需求名称 | 描述 |
|------|----------|------|
| N01 | 数据持久化 | 所有数据通过 SQLite 持久化存储 |
| N02 | 密码安全 | 密码使用 bcrypt 哈希存储 |
| N03 | XSS 防护 | 前端渲染使用 HTML 转义 |
| N04 | SQL 注入防护 | 所有 SQL 使用参数化查询 |
| N05 | 上传安全 | 文件类型白名单 + 文件大小限制 + 随机文件名 |
| N06 | 会话安全 | Cookie 设置 httpOnly，防 XSS 窃取 |

## 4. 数据约束

| 字段 | 约束 |
|------|------|
| 清单标题 | 必填 |
| 清单内容 | 必填，至少 1 条 |
| 分类 | 必填，必须为存在的分类 |
| 留言昵称 | 必填，不超过 50 字 |
| 留言内容 | 必填，不超过 2000 字 |
| 用户名 | 2-30 个字符 |
| 密码 | 至少 4 个字符 |
| 图片 | 5MB 以内，仅 JPG/PNG/GIF |
