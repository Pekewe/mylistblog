# 产品概述

> 版本：1.0.0 | 最后更新：2026-05-24

---

## 产品定位

「发布清单总站」是一个轻量级的清单内容管理与分享平台，专注于让用户能够方便地发布、浏览和管理各类主题清单。   
**（其实是看到了一些老东西想把它复现出来回忆一下，反正现在AI这么方便。）**

## 目标用户

| 用户角色 | 描述 | 核心需求 |
|---------|------|----------|
| 访客（浏览者） | 访问站点的普通用户 | 浏览清单、搜索内容、留言交流 |
| 发布人 | 经管理员授权的清单发布者 | 发布/管理自己的清单 |
| 管理员 | 站点运营者 | 审核内容、管理系统、配置站点 |

## 产品背景

个人站点形式的清单分享社区。管理员作为运营者，可以手动创建发布人账号；发布人通过账号验证提交清单内容；访客无需登录即可浏览和留言。三者权限分明。

## 核心价值

1. **简单直接** — 无需注册即可浏览，发布流程简洁
2. **轻量部署** — Node.js + SQLite 单文件数据库，无需复杂运维
3. **内容可控** — 管理员审核机制确保内容质量
4. **低门槛维护** — 纯 JS 栈，前后端统一语言
