## 当前版本（v1.0.0）功能覆盖

### 已实现

- [x] 清单 CRUD（前后台）
- [x] 发布人账号系统（管理员创建）
- [x] 发布权限控制（账号密码验证）
- [x] 留言提交 + 管理员审核 + 回复
- [x] 分类管理
- [x] 图片上传 + 外链
- [x] 商品编号字段
- [x] 后台搜索
- [x] 前三/后台分页
- [x] 站点设置

### 明确不实现

- ❌ 用户自主注册（发布人由管理员创建）
- ❌ 清单评论（仅保留留言板）
- ❌ 数据统计（非核心需求）
- ❌ 动态字段管理系统（增减字段直接改代码）

## 后续规划

| 功能 | 预估工时 | 说明 |

| 后台分页器优化 | 0.5天 | 分页控件样式改进 |
